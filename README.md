# SongBird — Music Video Maker

Небольшая программа для сборки музыкальных клипов и слайд-шоу под аудиотреки.
Десктопное приложение на PySide6.

## Что умеет

- **Клип** — склейка видеофрагментов с плавными переходами, зацикливание под длину трека, наложение аудио с fade in/out.
- **Слайд-шоу** — из картинок с эффектами (Ken Burns, zoom, pan) и частицами.
- **Генерация видео** — через Grok (xAI) по своему API-ключу.
- Склейка нескольких аудиотреков в одну композицию.

## Запуск

```
pip install PySide6 requests
python main.pyw
```

Нужен **ffmpeg** в `PATH` (или положите `ffmpeg.exe` / `ffprobe.exe` в папку `bin/`).

## Как пользоваться

Программа собирает видео под музыкальный трек. Слева — вкладки, основные две:

**Клип** — для видеоряда из готовых видеофрагментов:
1. Добавьте один или несколько аудиотреков (они склеятся в одну композицию).
2. Добавьте короткие видеофрагменты (например, 10-секундные из нейросети).
3. Нажмите сборку — программа склеит фрагменты с плавными переходами, зациклит их под длину трека и наложит музыку с плавным появлением/затуханием. На выходе — готовый MP4.

**Слайд-шоу** — если видеофрагментов нет, только картинки:
1. Добавьте трек и несколько картинок.
2. Программа покажет их по очереди с эффектами (плавное приближение, панорама) и частицами под музыку.

**Генерация видео** — сделать видеофрагменты прямо из программы (нужен ключ Grok, см. ниже).
**YouTube** — заготовка названия, описания и тегов для ролика.

Идея простая: трек длинный, а видеоряд короткий — он просто зацикливается. Под музыку этого достаточно.

## Ключ Grok

Для генерации видео вставьте свой ключ xAI на вкладке «Генерация видео» — он сохраняется локально в `state.json` (этот файл в `.gitignore` и в репозиторий не попадает). Где взять ключ: console.x.ai → API Keys.
